# Игра "Виселица" на Python 🎮

Простая консольная игра, где нужно угадать случайное слово из файла.

## Особенности
* 🎲 **Случайные слова** из внешнего файла `words.txt`.
* 💡 **Система подсказок**: введите `?`, чтобы открыть букву (цена: 1 жизнь).
* 🛡️ **Защита от ошибок**: проверка на ввод цифр, символов и повторных букв.

## Как запустить
1. Убедитесь, что у вас установлен Python.
2. Скачайте файлы `main.py` и `words.txt`.
3. Запустите игру командой:
   `python main.py`

## Управление
* Вводите одну русскую букву.
* Нажмите `?` для получения подсказки.
